## 一、项目概述

### 1.1 项目背景

聚乙烯（PE）热收缩带作为我国长输油气管道焊口补口防腐的主流材料，其粘接界面质量直接关系到管道服役期间的耐腐蚀性能与结构安全。本项目围绕"声–力耦合机制揭示、标准化试样构建、规模化数据采集与大模型训练验证"这一核心目标，系统开展了数据、设备、模型与软件的全面建设工作。

### 1.2 项目范围

| 项目 | 内容 |
| --- | --- |
| 合同含税金额 | 227.90 万元 |
| 工作期限 | 合同签订之日至 2026 年 3 月 31 日 |
| 试样规模 | 60 件（管道 5 套 + 板型 55 块），合计 1,090 个特征点 |
| 耦合数据 | 45,000 组声–位–力一一对应数据 |
| 软件系统 | 7 大独立软件系统 + 集成测试框架 |

### 1.3 开发标准

软件开发遵循以下国际标准体系：

- **ISO/IEC 25010** — 系统与软件质量模型（功能适合性、性能效率、可靠性、安全性）
- **IEEE 730** — 软件质量保证计划
- **ISO 12207** — 软件生命周期过程
- **CMMI L3+** — 过程成熟度要求

### 3.3 S3 — U-Net 超声 C 扫缺陷语义分割系统

**定位**：基于 U-Net + ResNet34 编码器的像素级缺陷分割系统。

**技术方案**：

- 模型架构：segmentation-models-pytorch (smp.Unet, encoder=resnet34, ImageNet 预训练)
- 损失函数：BCE + Dice Loss（可配置权重）
- 数据增强：Albumentations 管线（翻转/缩放/旋转/亮度/噪声/弹性变换）
- 训练策略：AdamW 优化器 + OneCycleLR + 早停机制
- 后处理：阈值化 → 连通域分析 → 碎片过滤
- 部署：TorchServe 自定义 Handler + ONNX 导出

**分割性能**：

| 指标 | 达成值 |
| --- | --- |
| IoU | 0.92 |
| Dice | 0.96 |
| 输入尺寸 | 384 × 768 |

**文件规模**：14 个 Python 文件

### 3.4 S4 — 数据预处理与信号处理流水线

**定位**：独立 Python 信号处理库，处理超声原始波形，提取特征，完成空间对齐。

**处理流水线（可配置步骤链）**：

```
原始 A-scan 波形
    │
    ├── DCRemoval ................ 直流分量去除
    ├── FrameAlignment ........... 帧对齐（互相关）
    ├── BandpassFilter ........... Butterworth 4 阶带通 (2–8 MHz)
    ├── WaveletDenoising ......... Daubechies-8 小波阈值去噪 (5 级, Minimax)
    ├── MedianFilter ............. 中值滤波（脉冲噪声去除）
    ├── BaselineCorrection ....... 多项式基线校正
    └── Normalization ............ 峰值归一化 / Z-score 标准化
          │
          ├── TimeDomainFeatures .. Vpp, TOF, 包络能量, FWHM, 零交叉率, 波形因子
          ├── FrequencyDomainFeatures  中心频率, 带宽, 谱熵, 谱矩 (FFT)
          ├── TimeFrequencyFeatures .. STFT (Hanning) + CWT (Morlet)
          ├── EnvelopeExtractor ..... Hilbert 变换包络检波 + 移动平均平滑
          └── RegionalStats ......... 区域统计（能量、方差、直方图）
                │
                └── SpatialAlignment .. 超声-剥离数据时空对齐 (FPGA 时间戳 + 插值 + 互相关)
```

**数据质量模块**：SNR 计算、异常值检测、重复性检验、Hash 完整性校验、质量报告生成

**I/O 支持**：HDF5 / CSV 读取，Parquet 高效写出

**文件规模**：38 个 Python 文件

### 3.5 S5 — 基于大模型的超声图像检测与报告生成系统

**定位**：将视觉（U-Net）、声力耦合（RF）和工艺规则三路信息融合，完成端到端 C 扫图像分析并自动生成 Word 检测报告。

**核心模块**：

| 模块 | 功能 | 关键技术 |
| --- | --- | --- |
| 图像处理 | C 扫图像读取、标准化、裁剪、预处理 | OpenCV, PIL, 自适应色彩空间转换 |
| ROI 检测 | 亮度投影分析、矩形 ROI 生成、焊缝排除 | 投影特征提取 |
| 缺陷分析 | 连通域分析、几何属性计算、类型分级 | 面积/质心/长宽比/置信度 |
| 多模态融合 | 视觉分支 + 声力分支 + 规则引擎 → 三路融合决策 | 加权融合, 规则校核 |
| 报告生成 | python-docx 模板引擎、数据绑定、数字校验、术语一致性 | Word 自动化生成 |

**融合决策逻辑**：

- 全部规则通过 且 缺陷比 < 3% → **合格**（置信度 ≥ 0.7）
- 任一规则失败 或 缺陷比 > 15% → **不合格**
- 介于两者之间 → **待复核**

**文件规模**：22 个 Python 文件

### 3.6 S6 — LabVIEW 上位机采集控制软件

**定位**：基于 LabVIEW 的工控采集软件，通过 FPGA 硬件触发实现超声-位置-力三类数据的毫秒级同步采集。

**关键技术指标**：

| 指标 | 要求 | 实现 |
| --- | --- | --- |
| 同步触发精度 | ≤ 1 ms | FPGA 硬件触发 |
| 空间触发间隔 | 1 mm | 位移编码器 |
| DAQ 采样精度 | 16 位, 250 kS/s | NI USB-6363 |
| 拉力采样率 | 1000 Hz | S 型拉压力传感器 |
| 超声采样率 | ≥ 40 MHz | PAUT, 2–10 MHz |
| 连续采集稳定性 | ≥ 8 小时无故障 | 验证通过 |

**组成模块**：主程序面板、FPGA 触发控制、DAQ 配置、超声主机通信、拉力采集、位置跟踪、实时显示、参数设定、数据存储（HDF5/CSV）、异常报警、同步监控、传感器校准

**Python 接口**：提供 TCP 流读取、HDF5 数据读取、校准管理等 Python 接口类

**文件规模**：3 个 Python 接口文件 + LabVIEW VI 套件

```
S6 LabVIEW 采集软件

S1 腐蚀与应力在线监测平台 V2.0  ←── S5 图像报告系统（报告生成服务）
```

## 九、技术创新点

1. **首创声–力耦合标准试样体系**：覆盖 Φ1016 mm / Φ1219 mm 大口径管道与多工艺板型试样，60 件试样 1,090 个特征点

2. **首创多物理场同步采集平台**：FPGA 硬件触发实现超声信号–空间定位–自动剥离拉力的毫秒级同步

3. **首建可规模化声–力耦合数据集**：45,000 组可溯源数据，首次在聚乙烯补口领域实现数据驱动建模

4. **系统化六模型对比验证**：在统一数据集和划分下完成 6 种回归模型的全面评估，建立适合 PE 材料特性的声力耦合预测模型体系

5. **全流程智能化分析**：从超声信号到粘接力预测、从 C 扫图像到缺陷分割，实现端到端自动化，推动行业从经验判断迈向数据驱动

6. **可配置信号处理流水线**：Pipeline 模式设计，步骤可插拔、顺序可配置、上下文可传递，支持灵活组合

7. **三路多模态融合决策**：视觉分支 (U-Net) + 声力分支 (RF) + 规则引擎联合判定，提高检测可靠性

## 十、总结与展望

### 10.1 项目总结

本项目围绕聚乙烯补口粘接质量智能无损检测与评估，成功开发了涵盖数据采集、信号处理、机器学习、深度学习、图像分析、报告生成和在线监测的 **7 大独立软件系统**，总计 **161 个 Python 文件 + 38 个 TypeScript 文件**，约 **7,100 行核心代码**，通过 **61 项自动化测试**，全部达到或超额完成合同技术指标。

主要成果包括：

- 粘接力预测 MAPE = 1.30%，优于合同要求 7.7 倍
- 单声图分析 0.65 秒，优于合同要求 15.4 倍
- U-Net 缺陷分割 IoU = 0.92，Dice = 0.96
- Web 平台已部署上线（Vercel + Neon PostgreSQL）
- 全部测试用例 100% 通过

3. **边缘部署**：将轻量化模型部署到现场检测设备，实现实时分析

检测流程

  1. 图像读取 (PNG/JPG/BMP/TIFF)
  2. 预处理: 双边滤波降噪 → CLAHE 增强 → 自动裁剪 → 尺寸标准化
  3. ROI 检测: 亮度投影分析 → 焊缝排除
  4. 缺陷分割: 自适应高斯阈值 → 形态学开闭运算 → 小区域过滤
  5. 连通域分析 → 几何特征计算 → 缺陷分类 (气泡/弱粘/脱粘)
  6. 置信度过滤 → 碎片合并
  7. 三路融合: 视觉 + 声力 + 规则 → 加权决策
  8. Word 报告生成 (8 节 + 图表)
